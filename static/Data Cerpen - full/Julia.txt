Liang pisan reraman tiangé. Saking dumun, mapangapti mangda I Julia nyak ngranjing agama Hindu. Ipun sakadi jejeh nyarengin agama Hindu santukan nénten sumeken pisan polih paplajahan indik Hindu, punapi malih indik tatakramanring Bali. Sané dibi rauh sewala patra ipunné katiba ring reraman tiangé. Nyantenang déwék jagi jati-jati ngranjing Hindu. Julia punika makarya ring Amerika. Reraman tiangé nganikayang, Julia punika dados bintang filem. Katah sampun filem-filem sané kasarengin samaliha dados pemucuk ring filem. Wantah sampun ipun sané mintangin pastika jadi polih piala Oscar, Piala sané dados rerebutan sang waged ring filem. Ring duranagara utawi ring Amerika, sami uning ring I Julia. Dados bantang bebaosan kabecikan ipuné dados bintang filem.
